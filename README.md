# Mobile Calculator App 📱

This is a simple calculator I made using Python and Flask. 
It works on both mobile and desktop. The UI is dark themed because I like it that way.

### What it does
- Basic math: +, -, *, /, %, .
- Shows error if you divide by zero
- Saves your last 5 calculations so you don't forget
- Clean buttons, works without reloading the page
